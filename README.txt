===================================================
  GIS EPD Slot Grabber — Инструкция по установке
===================================================

ЧТО ДЕЛАЕТ ПРОГРАММА:
  Автоматически бронирует слоты в ГИС ЭПД в 10:00 и 12:00
  каждый день. Использует ваш Яндекс браузер для авторизации.

---------------------------------------------------
ШАГ 1 — Первый запуск (проверка)
---------------------------------------------------
1. Войдите в eopp.epd-portal.ru через Яндекс Браузер
   (обычный вход через Госуслуги)

2. Дважды щёлкните GisEpdGrabber.exe
   — откроется чёрное окно
   — программа проверит подключение к ГИС ЭПД
   — через 5-10 секунд закроется

3. Проверьте лог:
   %APPDATA%\GisEpdGrabber\grabber.log
   (Открыть: Win+R → %APPDATA%\GisEpdGrabber)

---------------------------------------------------
ШАГ 2 — Настройка автозапуска
---------------------------------------------------
1. Щёлкните правой кнопкой по setup_scheduler.bat
2. Выберите "Запустить от имени администратора"
3. Дождитесь сообщения "Задачи созданы!"

После этого программа будет запускаться автоматически
в 10:00 и 12:00 каждый день.

---------------------------------------------------
ВАЖНО — когда нужно войти заново
---------------------------------------------------
Если в логе появится "Сессия истекла":
1. Откройте Яндекс Браузер
2. Зайдите на eopp.epd-portal.ru
3. Войдите через Госуслуги как обычно
4. Закройте браузер — готово

Это нужно делать примерно раз в неделю.
