# GIS EPD Slot Grabber

Автоматическое бронирование слотов в [ГИС ЭПД](https://eopp.epd-portal.ru) в 10:00 и 12:00 МСК.

## Как работает

1. Читает куки Яндекс Браузера (или Edge/Chrome) прямо с вашего компьютера
2. Авторизуется на eopp.epd-portal.ru от вашего имени  
3. Находит активные заявки и бронирует слоты
4. Пишет результат в лог (и в Telegram, если настроен бот)

## Установка

### Шаг 1 — Скачать

Скачайте из [Releases](../../releases/latest):
- `GisEpdGrabber.exe` — основная программа
- `setup_scheduler.bat` — настройка автозапуска

### Шаг 2 — Войти в портал

Откройте Яндекс Браузер и войдите на `eopp.epd-portal.ru` через Госуслуги как обычно.

### Шаг 3 — Настроить автозапуск

Положите оба файла в одну папку.  
Щёлкните правой кнопкой по `setup_scheduler.bat` → **Запуск от имени администратора**.

Всё — программа будет запускаться автоматически в 10:00 и 12:00.

## Telegram уведомления (опционально)

Создайте бота через [@BotFather](https://t.me/BotFather) и установите переменные среды:

```
GIS_TG_TOKEN=токен_бота
GIS_TG_CHAT=id_чата
```

Или отредактируйте в `setup_scheduler.bat` строки с `set GIS_TG_TOKEN=...`.

## Когда нужно обновить сессию

Если в логе появится `Сессия истекла`:
1. Откройте Яндекс Браузер
2. Зайдите на `eopp.epd-portal.ru`
3. Войдите через Госуслуги
4. Закройте браузер

Это нужно примерно раз в неделю.
